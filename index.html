Technical Work

✦ FINAL YEAR PROJECT · 2025–2026
Weather & Events Alert App

A Flutter mobile application providing hyper-local weather updates for villages and cities, combined with a personal event reminder system — making it a complete daily planning companion.

Core Features

  • Weather Module — Real-time local weather by city/village with 7-day forecast, temperature, humidity, wind, and extreme weather alerts.
  • Event Module — Create, edit, and delete personal reminders with push notification alerts delivered via OneSignal.
  • User Module — Secure login/signup with Firebase Authentication, profile management, and personalized data per user.
  • Hyper-local Focus — Searchable dropdown for villages/cities including rural areas underserved by major weather platforms.
